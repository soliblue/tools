# Majorelle Tools

Majorelle theme assets for:

- VS Code
- iTerm2
- Oh My Zsh

## Contents

- `vscode/`: local VS Code theme extension with `Majorelle`, `Majorelle Medium`, and `Majorelle Light`
- `iterm/`: matching `.itermcolors` presets
- `oh-my-zsh/`: matching shell prompt theme

## Notes

- `CLAUDE.md` documents the setup details for agents
- `AGENTS.md` is a symlink to `CLAUDE.md`
