# Majorelle Tools

Majorelle theme assets and setup notes for:

- VS Code
- iTerm2
- Oh My Zsh

This repo is a small glimpse into the author's personal editor and terminal setup.

## Contents

- `vscode/`: local VS Code theme extension with `Majorelle`, `Majorelle Medium`, and `Majorelle Light`
- `iterm/`: matching `.itermcolors` presets
- `oh-my-zsh/`: matching shell prompt theme

## Theme Variants

| Variant | UI Theme | Best For |
|---------|----------|----------|
| Majorelle | Dark | Late-night coding |
| Majorelle Medium | Dark | Softer indigo background |
| Majorelle Light | Light | Warm off-white background |

## Notes

- `CLAUDE.md` documents the setup details for agents
- `AGENTS.md` is a symlink to `CLAUDE.md`
- Recommended local VS Code behavior: bind `cmd+t` to open a new terminal in the editor area as a new editor tab
